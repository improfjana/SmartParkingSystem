Smart Parking System 🚗

Overview

Smart Parking System is a Java-based application developed to simulate a real parking management system. It allows users to manage vehicles, parking spaces, parking tickets, and parking fees through an organized and user-friendly system.

Features

* Add and remove vehicles
* Support multiple vehicle types (Car, Truck, Motorcycle, Electric Car)
* Generate parking tickets
* Calculate parking fees based on parking duration
* Manage parking floors and parking spots
* Simple graphical user interface (GUI)
* File handling for saving and loading data

Technologies Used

* Java
* JavaFX
* Object-Oriented Programming (OOP)
* File Handling
* Exception Handling
